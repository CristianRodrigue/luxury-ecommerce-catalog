# Luxury E-Commerce Catalog Prototype

A high-end, cinematic e-commerce landing page built with **Next.js 15+**, **Framer Motion**, and **Tailwind CSS**. This project demonstrates premium UI/UX design principles, featuring a sophisticated "Dark Mode" aesthetic and fluid interactive components.

## 🚀 Key Features

- **Cinematic Hero Experience:** Background video integration with smooth text overlays.
- **Interactive Product Catalog:** Dynamic grid with filtering capabilities (Staples, Limited, Combat Gear).
- **Premium Animations:** High-performance micro-interactions using Framer Motion.
- **Modern Tech Stack:** Built with the latest Next.js App Router for optimal performance.
- **Responsive & Accessible:** Fully optimized for all screen sizes with a focus on luxury brand presentation.

## 🛠️ Technologies Used

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **State & Logic:** React 19 (Hooks/Suspense)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons & Typography:** Premium Serif & Sans-Serif font pairings.

## 📦 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/CristianRodrigue/luxury-ecommerce-catalog.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
